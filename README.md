# Proyecto
el hada y la ballena, proyecto de creación

## Descripción
Web del proyecto creativo el hada y la ballena

### Prerequisitos
* NodeJS
* NPM

### Instalación
Instalar dependencias de node
```
npm install
```

### Tareas de desarrollo
Arranca generador de estáticos y servidor
```
npm run start
```

#### Construído con
* HTML 
* CSS 
* Javascript
