# Proyecto
el hada y la ballena, proyecto de creación

## Descripción
Web del proyecto creativo el hada y la ballena

### Prerequisitos
* NodeJS
* NPM

### Instalación
Instalar dependencias de node
```
npm install
```

### Arrancar servidor http
```
npm run server
```

### Tareas de desarrollo
Tarea que observa cambios para compilar archivos.
```
npm run watch
```

Tarea que prepara la carpeta dist para su distribución
```
npm run build


#### Construído con
* HTML 
* CSS 
* Javascript
